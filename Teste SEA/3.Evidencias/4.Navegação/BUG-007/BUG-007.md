# BUG-007 — Sidebar: clique nos ícones não redireciona para a tela "Em breve"

## Tipo
Navegação / Conformidade com Protótipo

## Severidade
Alta

## Ambiente
- SO: Windows 11
- Microsoft Edge: Versão 143.0.3650.96 (64 bits)
- Google Chrome: Versão 143.0.7499.170 (64 bits)

## Passos para reproduzir
1. Acessar a aplicação.
2. Clicar em qualquer ícone do menu lateral (sidebar).
3. Repetir o teste clicando em outros ícones.

## Resultado atual (Aplicação)
- Ao clicar nos ícones do menu lateral, não ocorre redirecionamento (nenhuma navegação é executada).

## Resultado esperado (Protótipo)
- Ao clicar em qualquer item do menu lateral, o usuário deve ser direcionado para a tela "Em breve", contendo:
  - página com conteúdo em branco;
  - uma barra/box azul no topo;
  - texto "Em breve" centralizado.

## Impacto
- Quebra a navegação principal: itens do menu lateral não redirecionam para "Em breve".
- Impede validar o requisito do desafio sobre redirecionamento de links.
- Bloqueia exploração/teste de telas e fluxos dependentes do menu.
