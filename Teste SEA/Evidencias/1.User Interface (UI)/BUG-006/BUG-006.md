# BUG-006 — Sidebar: hover não exibe destaque (retângulo branco) conforme protótipo

## Tipo
UI / Conformidade com Protótipo 

## Severidade
Baixa  

## Ambiente
- SO: Windows 10/11
- Microsoft Edge: Versão 143.0.3650.96 (64 bits)
- Google Chrome: Versão 143.0.7499.170 (64 bits)

## Passos para reproduzir
1. Acessar a aplicação.
2. Passar o mouse sobre os ícones do menu lateral (sidebar).

## Resultado atual (Aplicação)
- Ao passar o mouse sobre os ícones, não aparece o retângulo branco de destaque ao lado esquerdo do ícone.

## Resultado esperado (Protótipo)
- Ao passar o mouse sobre um ícone da sidebar, deve aparecer um retângulo branco de destaque no lado esquerdo do ícone, conforme protótipo.

## Impacto
- Reduz o feedback visual para o usuário ao navegar no menu lateral, dificultando identificar qual item está sendo apontado.
- Quebra a conformidade com o protótipo, impactando consistência e percepção de qualidade da UI.
